# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes Superstore sales data to uncover business insights using Python, SQL, and Power BI.

The objective is to answer important business questions such as:

- Which product category generates the highest sales?
- Which category earns the highest profit?
- Which region contributes the most revenue?
- Which customer segment drives the business?
- Which products perform the best?

---

## 🛠 Tools Used

- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Power BI
- Excel

---

## 📂 Project Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. Business KPI Calculation
4. SQL Business Analysis
5. Interactive Dashboard in Power BI

---

## 📈 Key Business Insights

- Technology contributes the highest share of sales.
- Technology also generates the highest overall profit.
- The West region contributes the largest percentage of sales.
- Consumer segment accounts for over 50% of total sales.
- Several products consistently outperform the rest in revenue.
